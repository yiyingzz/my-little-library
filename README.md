# my little library

A book-tracking app made as part of the [full-stack JavaScript curriculum](https://www.theodinproject.com/courses/javascript/lessons/library) at [The Odin Project](https://www.theodinproject.com).

## Functionality
Users are able to:
- Add new books and mark them as in progress (reading) or completed (read)
- Toggle "reading" or "read" status on books
- Data is saved in local storage

## Made with
- HTML & CSS (Sass)
- JavaScript
- mobile-first design
